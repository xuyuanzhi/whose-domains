How to Transfer a Domain

Complete Guide to Domain Transfers

Domain transfer is the process of moving a domain name from one registrar to another. Understanding the transfer process helps ensure a smooth transition without downtime or loss of services.

Prerequisites for Transfer

  • Domain must be at least 60 days old (ICANN rule)
  • Domain must not have been transferred in the last 60 days
  • Domain must be unlocked (clientTransferProhibited removed)
  • You must have access to the admin email address on file
  • Domain must not be expired (or in redemption period)
  • WHOIS privacy may need to be temporarily disabled

Step-by-Step Transfer Process

Step 1: Prepare at the Current Registrar

  1. Unlock the domain (disable transfer lock)
  2. Disable WHOIS privacy if required
  3. Obtain the authorization code (EPP code / transfer key)
  4. Verify the admin contact email is correct and accessible

Step 2: Initiate at the New Registrar

  1. Start a transfer request at the new registrar
  2. Enter your domain name and authorization code
  3. Pay the transfer fee (usually includes a 1-year extension)

Step 3: Confirm the Transfer

  1. Check your admin email for a transfer confirmation request
  2. Approve the transfer (usually via a link in the email)
  3. The old registrar may also send a notification — don't reject it

Step 4: Wait for Completion

  • Transfers typically complete within 5-7 days
  • Some registrars offer expedited transfers
  • DNS settings are usually preserved during transfer

Common Transfer Issues

  • Transfer rejected: Domain may still be locked, or EPP code is incorrect
  • Email not received: Check spam, or update admin email first
  • 60-day lock: Recently registered or transferred domains cannot be moved
  • Expired domain: Must renew before transferring
  • DNS disruption: Copy all DNS records to the new registrar before transferring

Tips for a Smooth Transfer

  • ✔ Back up all DNS records before initiating the transfer
  • ✔ Set up the same DNS records at the new registrar in advance
  • ✔ Don't let the domain expire during the transfer process
  • ✔ Keep the old registrar account active until transfer completes
  • ✔ Re-enable transfer lock and WHOIS privacy after the transfer
