    How Domain Scoring Works

    Our scoring algorithm evaluates five critical dimensions of domain health: Domain Age (15%) — older domains are more trusted; SSL/HTTPS (25%) — encryption and HSTS configuration; Email Security (20%) — SPF, DKIM, and DMARC records; Security Headers (20%) — XSS protection, CSP, and more; DNS Configuration (20%) — nameserver redundancy, IPv6, DNSSEC. Each dimension is scored 0-100 and weighted to produce a final grade from A+ to F.
